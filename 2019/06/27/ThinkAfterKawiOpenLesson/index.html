快手公开课后的感悟

发表于 2019-06-27

今天很荣幸在公司通过直播观看了《快手公开课》，公开课上面有很多利用快手平台在赚钱的老铁介绍了自己的一些经验，感觉收获颇为丰厚。特地在这篇文章中做一个感悟总结。

摒弃固有认知

曾经的快手在一线城市和部分小资产阶级人群中是口碑极其不好的。然而在下沉市场当中，快手是非常厉害的。2亿的DAU带来的是一个非常巨大的市场。对于从事IT行业的人员来说，可能就会忽视这里面的巨大的价值。

在听第一位嘉宾演讲的时候，我就有了非常大的震撼。第一位嘉宾是一对初中毕业就不读书的夫妻，他们在快手上面拥有1100万粉丝。然后他们通过电商实现了财务自由，夫妻两个人还都不到30岁。当时听到的确是非常震惊的，因为在我们固有的认知当中，尤其现在互联网的透明化，很多不努力读书的孩子的路已经越走越窄了。但是快手给与了他们一个可以被关注的机会，带给了他们巨大的流量，并且利用这么大的流量实现了生活的飞跃进步。

相信他们在运营快手账号的时候，肯定也是付出了诸多的心血，风光的背后一定也有他们不为人知的艰辛。但是互联网的风口成功地带来了巨大的流量，带来了一些与众不同的商机。

什么最重要

对于程序员来说，相信技术一定是我们认为最重要的东西。换到其他赛道，其实我们的技术就像制作了商品的工艺，本质上像是一种工具。工具重要吗？非常重要，因为没有这些工具，就没有办法生产出具体的产品。没有产品，自然是无从谈及销售和运营。所以技术是制造产品的根本，如果没有技术，就完全不会有产品。

然而技术只能算是产品的躯壳和实现，技术本身是没有灵魂的。最终发挥了巨大商业价值的，是生产出的产品。比如在快手上面拍视频，用到的软件和拍摄工具都属于技术，然而最终生产出的视频内容才是产品。对于用户来说，没有人关注这条视频怎么拍的，只在乎这条视频能不能满足我开心欢乐的目的，或者满足我了解其他东西的目的。从这个角度出发，精心经过技术加工能够使得视频变得更好，从而产生了更为优质的内容。

引申到软件开发也是一样的，某些产品的实现是依赖技术的，没有技术完全做不出来。然而对于大部分游戏来讲，其实需要的技术很多人已经满足开发条件了。在这个情况下，其实比拼的更多的是策划和产品的能力。比如《太吾绘卷》这样的游戏，很多人肯定也是可以靠自己做出来一个一样的游戏的(单纯的技术拷贝)。然而《太吾绘卷》的策划灵魂，优雅的文笔，以及沙盒式的玩法，则是其他人想不到的。当然《太吾绘卷》也有作者巨大的坚持，从而取得了巨大的收益。

所以技术是程序员的立命之本，但是如果想要有更高更好的发展，除了在技术上面需要投入巨量的时间和精力之外，也一定要有思考产品的能力。如果没有产品的想法，就会沉浸在技术之中，忽视掉一些潜在的商业机会。对待事物，可以按照一个产品的方向进行理解，努力地去思考内部的商业运转规律和运营法则，以及它所产生的价值。对待新鲜事物或者自己不懂的事物不应该排斥，不应该拒绝，而是更应该努力拥抱。

像运营产品一样运营自己

以技术谋生的程序员，也可以像运营产品一样来运营自己。应该多多地去挖掘周围的机会，而不是一味地沉浸在代码天地中，也许能够找到一条属于自己的道路。

我的这个想法也是在读了《软技能：代码之外的生存指南》后产生的。在这里也是强烈地推荐一下这本书，能够扭转很多程序员固有的认知。比如技术无敌论等等。这本书也介绍了写博客以及开拓人脉，多认识人的重要性，可以说对于程序员”肥宅”的认知有巨大的挑战。

商业价值

从社会角度来说，利用正当手段创造巨大的商业价值，不仅能够让自己的生活变得美好，也能让周围的人的生活变得更好。从商业价值的角度去思考问题，是一个很俗但是却不可避免的问题。说实话我是比较羡慕很年轻就实现财务的初步自由的人的，倒不是羡慕他们美好的生活，而是羡慕他们可以不受束缚地去做一些自己想做的事情。

如果有一天我也能够稍微不太受财务束缚了，能够让自己过得很好的话，我对独立游戏的热情肯定是永远都在的，也永远都希望能够做出更好的作品吧。毕竟做一个好游戏，是无与伦比的精神价值啊。

就像当初看到快手的创始人的情怀理念，对于产品的极度克制，都是令我极为钦佩的。我想我也是个有情怀的人吧，如果有一天我的这些愿望也能够不断实现的话，我也希望能够有机会用自己的技术转化更多更有价值的产品。

他们鼓舞了我，我也要继续保持前进！

# 快手 # 感悟
